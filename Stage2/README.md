## Покрываемые задачи ВКР

- **Задача 3.** Проектирование методики и архитектуры детектора аномальной активности (детальное проектирование).
- **Задача 4.** Разработка подсистемы предобработки и формирования обучающих и тестовых выборок (проектирование).
- **Задача 6.** Разработка конечно-автоматного агента активного тестирования (проектирование).

## Связь с Этапом 1

Все ключевые архитектурные решения опираются на результаты сравнительного анализа Этапа 1:

- Целевая нейросетевая архитектура — **CNN-LSTM** (4,60 баллов из 5 по семи критериям сравнения, см. [Stage1 §3.5](../Stage1/report_stage1.md#35-балльная-оценка-архитектур-по-критериям)).
- Основной датасет — **UNSW-NB15** (4,80 баллов из 5 по восьми критериям, см. [Stage1 §4.5](../Stage1/report_stage1.md#45-взвешенная-балльная-оценка-и-решение-по-выбору)).
- Вспомогательный датасет — **CICIDS2017** (3,75 баллов) для cross-evaluation.
- Архитектура активного тестирования — **templates + FSM + drift-injector** ([Stage1 §5.3](../Stage1/report_stage1.md#53-конечно-автоматный-подход-к-шаблонной-генерации)).
- Функциональные требования FR-1…FR-10 и нефункциональные NFR-1…NFR-9 — из [Stage1 §6](../Stage1/report_stage1.md#6-формализация-функциональных-и-нефункциональных-требований-к-системе).
