## Сборка

Требуется TeX-дистрибутив с XeLaTeX и Biber (TeX Live 2022+ или
MiKTeX) с пакетами `biblatex-gost`, `polyglossia`, `fontspec`,
`lastpage`. Системные шрифты Times New Roman, Arial, Courier New
(штатно установлены в Windows; на Linux/macOS поставить через
`fontspec`-совместимый набор или подменить альтернативами).

```powershell
cd Report
xelatex report.tex
biber   report
xelatex report.tex
xelatex report.tex
```

Результат: `report.pdf`.

Альтернатива (одной командой) через latexmk:

```powershell
latexmk -xelatex -interaction=nonstopmode report.tex
```

## Содержание этапов

### Этап 1. Изучение литературы и теоретический анализ
Покрывает задачи 1–4 Этапа 1 ВКР.txt: обзор IDS/IPS/NIDS, статистические
и DL-методы обнаружения аномалий, моделирование атак с ИИ (включая GAN),
сравнение 10 датасетов NIDS по 8 критериям, обоснование выбора
UNSW-NB15, метрики качества, формализация требований (10 FR + 9 NFR)
и трёх гипотез исследования.

### Этап 2. Анализ архитектур и проектирование методики
Покрывает задачи 1–3 Этапа 2 ВКР.txt: сравнение 9 архитектур
нейронных сетей с опубликованными NIDS-результатами и явной привязкой
к первоисточникам, обоснование выбора CNN-LSTM как proposed, схема
признаков UNSW-NB15, pipeline обработки (cleanup → encoding → log/
robust scaling → windowing), focal loss для балансировки классов,
библиотека из 7 шаблонов атак, FSM-планировщик (11 состояний),
drift-инжектор (3 типа), протокол шести экспериментов E1–E6.

### Этап 3. Разработка прототипа системы
Покрывает задачи 1–3 Этапа 3 ВКР.txt: структура программного проекта
`diploma_nids` (4230 строк Python, 9 тестовых модулей), 15 моделей
(10 нейросетевых + 5 классических) в едином registry, training
pipeline с focal loss и AdamW, evaluation pipeline с температурной
калибровкой и bootstrap-CI, реализация ИИ-злоумышленника
(шаблоны + FSM + drift-инжектор), FastAPI + Streamlit-интерфейс.

### Этап 4. Тестирование, оптимизация и оформление (основной раздел)
Покрывает задачи 1–4 Этапа 4 ВКР.txt: шесть экспериментов E1–E6.

| Эксп. | Содержание |
|---|---|
| E1 | Сравнение 12 моделей × до 3 seed (22 запуска): сводная таблица F1, PR-AUC, FPR, latency |
| E2 | Confusion matrix CNN-LSTM + полнота по 10 классам атак UNSW-NB15 |
| E3 | Калибровка температурой (ECE: 0.089 → 0.047) + threshold под target FPR = 0.01 |
| E4 | Стресс-тест с ИИ-злоумышленником (600 тиков, 16 412 записей, per-state recall) |
| E5 | Drift sweep (covariate × concept × 5 интенсивностей, 10 точек) |
| E6 | Latency для всех 12 моделей, соответствие NFR-3 |
| Demo | Демонстрационный прогон с timeline и алертами |
| Обоснование | Выбор CNN-LSTM по 11 критериям (сравнительная таблица) |
| Гипотезы | Сводное подтверждение/уточнение трёх гипотез |
