# Stage 1 — Аналитико-постановочный этап

Промежуточный отчёт по результатам первого этапа ВКР: аналитический обзор предметной области, формализация требований к системе и обоснование архитектурных решений.

## Покрываемые задачи ВКР

- **Задача 1.** Аналитический обзор предметной области:
  - сравнительный анализ классических, машинно-обучающихся и нейросетевых подходов;
  - анализ нейросетевых архитектур для обработки последовательностей потоковых признаков;
  - сравнительный анализ публично доступных датасетов NIDS;
  - систематизация подходов к активному тестированию NIDS.
- **Задача 2.** Формализация функциональных и нефункциональных требований к системе.
- **Задача 3.** Обоснование выбора целевой архитектуры детектора и принципов организации конечно-автоматного агента активного тестирования (на уровне предварительного обоснования; детальное проектирование — на Этапе 2).
